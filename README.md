# Taller Práctico Tkinter - Login App

Ejercicio de clase: Aplicación de inicio de sesión con Tkinter y conexión a JSON.

## Actividades Realizadas

1. Cambiar el usuario y la contraseña.
2. Agregar los datos en un JSON y luego consumirlo.
3. Modificar el tamaño de la ventana.
4. Cambiar el título del formulario.
5. Modificar los mensajes de éxito y error.
6. Agregar color de fondo a la ventana.
7. Agregar un botón Limpiar.
8. Validar que los campos no estén vacíos.
9. Agregar un segundo usuario válido.
10. Abrir una nueva ventana al iniciar sesión correctamente.
11. Reemplazar el uso de pack() por grid().
